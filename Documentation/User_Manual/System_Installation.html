System Installation

1. Configuring the virtual machine

All the above examples will use the Oracle VirtualBox virtual machine emulator. You can of course use another virtual machine emulator, like qemu or Bosch, but you will have to find by yourself how to configure it.

  1. Install and start Oracle VirtualBox.
  2. Create a new virtual machine called Lemon :
    • Type : Other
    • Version : Other/Unknown
  3. Choose a 16 MB RAM size.
  4. Create a new virtual hard disk :
    • Hard drive file type : VDI
    • Hard drive allocation mode : dynamic
    • Hard disk size : 64 MB
    The machine has been successfully created. It's time to configure it.
  5. Choose a video memory size of 2 MB.
  6. Click on the Serial Ports tab and enable the serial port 1.
  7. Configure the following parameters :
    • Port Number : COM1
    • Port Mode : Host pipe
    • Check the Create pipe box
    • Port/File Path : /tmp/Lemon_Serial_Port_Pipe
  8. ATTENTION : be sure that the Lemon image has the right hard disk driver for the hard disk controller you chose (IDE or SATA).

The virtual machine is now fully operational.

2. Installing the system

  1. Start the machine and use the Lemon_Installer_CD_Image.iso image when asked to insert a CDROM.
  2. The installer will start. Accept the warning message and choose the full hard disk installation type.
  3. Remove the CDROM when the installer tells you to do that and reboot the virtual machine.
  4. Lemon is now installed. Welcome !
